Unified marketing automation suite for outreach, design, and landing page generation.

<args>$ARGUMENTS</args>

## Modules

- `outreach` / `start` / `workflow` - Multi-platform outreach automation
- `canva` / `design` - Canva design & local file editing
- `lp` / `landing-page` - Landing page generation

## Workflow

1. **Parse Arguments** from `$ARGUMENTS`

2. **If No Arguments or Need Clarification**
   Use `AskUserQuestion` to gather context:

   **Question 1: Module Selection**
   ```
   header: "Module"
   question: "Which automation do you need?"
   options:
     - label: "Outreach Automation"
       description: "LinkedIn, Twitter, Instagram, Gmail campaigns with visual workflow"
     - label: "Canva & Design"
       description: "Canva API, file editing (PDF/PPTX/DOCX/XLSX)"
     - label: "Landing Page"
       description: "Generate landing pages (HTML/React/Next.js/Astro/Vue)"
   ```

3. **For Outreach Module** (`/10x outreach` or `/10x start`)
   Ask via `AskUserQuestion`:
   ```
   header: "Action"
   question: "What outreach action do you want?"
   options:
     - label: "Start Visual Canvas"
       description: "Launch TLDraw workflow designer on localhost:3000"
     - label: "Create Workflow"
       description: "Design multi-platform outreach sequence"
     - label: "LinkedIn Outreach"
       description: "Connect, message, engage on LinkedIn"
     - label: "Twitter Outreach"
       description: "Follow, DM, reply on Twitter/X"
     - label: "Instagram Outreach"
       description: "Follow, DM, comment on Instagram"
     - label: "Discover People"
       description: "Find targets with Exa AI"
   ```

4. **For Canva Module** (`/10x canva`)
   Ask via `AskUserQuestion`:
   ```
   header: "Action"
   question: "What design action do you need?"
   options:
     - label: "Browse Designs"
       description: "List and search Canva designs"
     - label: "Create Design"
       description: "Create new design in Canva"
     - label: "Export Design"
       description: "Export to PNG/PDF/MP4"
     - label: "Edit Local File"
       description: "Modify PDF/PPTX/DOCX/XLSX"
   ```

5. **For Landing Page Module** (`/10x lp` or `/10x landing-page`)
   Ask via `AskUserQuestion`:
   ```
   header: "Project"
   question: "What do you want to do?"
   options:
     - label: "New Project"
       description: "Start fresh landing page with multi-agent workflow"
     - label: "Resume Project"
       description: "Continue existing project"
     - label: "List Projects"
       description: "Show all landing page projects"
   ```

   **If New Project, ask:**
   ```
   header: "Tech Stack"
   question: "Which tech stack for your landing page?"
   options:
     - label: "HTML (Recommended)"
       description: "Static HTML/CSS/JS - no build step"
     - label: "React"
       description: "React 18 with Vite"
     - label: "Next.js"
       description: "Next.js 14 App Router"
     - label: "Astro"
       description: "Astro 4 - fast static sites"
     - label: "Vue"
       description: "Vue 3 with Vite"
   ```

6. **Execute Action**
   - Route to appropriate sub-skill based on selection
   - Activate `10x-team` skill
   - Load relevant module from `.claude/skills/10x-team/`
