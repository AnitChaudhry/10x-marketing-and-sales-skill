# /lp - Landing Page Shortcut

---

## Description

Shortcut for `/10x/landing-page`. Creates high-converting landing pages.

---

## Usage

```
/lp [action] [project-name]
```

---

## Prompt

Redirect to `/10x/landing-page` command.

Activate the landing page skill: `.claude/skills/10x-team/landing-page/SKILL.md`

Arguments: `$ARGUMENTS`

If no arguments, start a new landing page project.

---

## Examples

```
/lp                      # Start new project
/lp new                  # Start new project
/lp resume my-app        # Resume existing
/lp list                 # List all projects
```
