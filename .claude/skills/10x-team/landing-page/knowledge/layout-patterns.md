# Layout Patterns & Section Types

> **Used by**: Design Agent, Build Agent
> **When**: Structuring hero sections and page layouts

---

## Above-the-Fold (ATF) Layout Types

### 1. Full-Page Photo
Background image with text overlay.
```
Best for: Lifestyle brands, emotional appeal
Examples: OnX Hunt, Opendoor, Row7 Seeds
```

### 2. Full-Page Illustration
Custom illustration as hero visual.
```
Best for: Tech products, creative brands
Examples: Intercom, Raycast, Discord, Lemonade
```

### 3. Full-Page Video
Background video with text overlay.
```
Best for: Experiential brands, premium products
Examples: Origin, Charity:Water, OpenAI
```

### 4. Explosions
UI elements/screenshots "exploding" from center.
```
Best for: SaaS, apps with strong UI
Examples: Quip, Wealthsimple, Amie
```

### 5. Sweeping Layouts
Diagonal or curved compositions.
```
Best for: Modern tech, dynamic brands
Examples: Stripe, GitHub, Compound
```

### 6. Half-and-Half
50/50 split between text and visual.
```
Best for: Clear product showcase, balanced messaging
Examples: Oscar, Superlist, Untitled UI
```

### 7. Teach by Example
Show product solving a real problem.
```
Best for: Products with clear use cases
Examples: Basecamp, Learn UI Design, Olipop
```

### 8. Text-as-Imagery
Typography IS the visual element.
```
Best for: Personal brands, minimal designs
Examples: Jeremiah Shoaf, Simon Rogers
```

### 9. UI Screenshot
Clean product screenshot as hero.
```
Best for: SaaS, software products
Examples: Linear, Framer, Superhuman
```

### 10. Presented UI
Screenshot in device mockup or styled frame.
```
Best for: Apps, polished product presentations
Examples: Amie, Betterment, Pipe
```

## Section Layout Types (Below the Fold)

### 1. Text + Image + Visual Interest
Standard layout with decorative elements.

### 2. Grid / Exploded Grid
Features in 2-4 column grid layout.

### 3. Bento
Mixed-size grid boxes (magazine style).

### 4. Key Idea
Single focused message with supporting visual.

### 5. Timeline
Sequential steps or process visualization.

### 6. Globe/Map/Place
Geographic or location-based visual.

### 7. Full-Page Imagery
Section with dominant background image.

### 8. Punch Combos
Alternating left-right content blocks.

### 9. Explosion Layout
Content radiating from central element.

### 10. Media Cards
Content in contained card components.

### 11. Half-and-Half
50/50 content split (alternating sides).

### 12. Animation-based
Motion as primary visual interest.

### 13. Pricing Plans
Comparison cards for tiers.

### 14. Basic Blurb
Simple icon + heading + text grid.

### 15. Image-as-Background
Text over section background image.

### 16. Highlighted Phrases
Typography-focused with emphasis styling.

### 17. Tweet Grid
Social proof in card format.

### 18. XL Layout
Oversized typography or visuals.

## Layout Selection Guide

| Product Type | Recommended ATF | Section Style |
|--------------|-----------------|---------------|
| SaaS/Software | UI Screenshot, Explosions | Grid, Half-and-Half |
| Mobile App | Presented UI, Device Mockup | Cards, Timeline |
| E-commerce | Full-Page Photo | Bento, Media Cards |
| Service/Agency | Full-Page Video/Photo | Punch Combos |
| Course/Info | Teach by Example | Timeline, Key Idea |
| Premium/Luxury | Full-Page Photo/Video | XL Layout, Minimal |
