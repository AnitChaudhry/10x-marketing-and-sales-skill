## By Use Case

### "I want to write content"
1. Read [04-content-creation.md](./kit-guide/04-content-creation.md)
2. Research keywords: `/seo "keyword research"`
3. Write: `/content:blog "topic"` or `/content:good "request"`
4. Optimize: `/content:cro "optimize for conversions"`

### "I need to create social media content"
1. Define strategy in [05-social-media.md](./kit-guide/05-social-media.md)
2. Create content calendar
3. Generate posts: `/social twitter post`, `/social linkedin post`, etc.
4. Track performance

### "I'm launching a marketing campaign"
1. Plan in [06-campaign-management.md](./kit-guide/06-campaign-management.md)
2. Create campaign: `/campaign create "campaign name"`
3. Create content across channels
4. Analyze: `/campaign analyze`

### "I need beautiful visuals"
1. Reference [02-image-generation.md](./kit-guide/02-image-generation.md)
2. Create design: `/design:good "design description"`
3. Generate images: `/design:generate "image description"`
4. Optimize: Use scripts in [09-scripts-reference.md](./kit-guide/09-scripts-reference.md)

### "I want to rank in search results"
1. Read [07-seo-optimization.md](./kit-guide/07-seo-optimization.md)
2. Research keywords: `/seo "keyword research"`
3. Create content: `/content:blog "topic"`
4. Optimize: `/seo "optimize page"`
5. Track: Monitor rankings

### "I'm setting up the kit from scratch"
1. Follow [08-project-setup.md](./kit-guide/08-project-setup.md)
2. Read [01-brand-system.md](./kit-guide/01-brand-system.md) to setup brand
3. Create first piece of content
4. Monitor and iterate
